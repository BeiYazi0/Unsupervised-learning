# 无监督学习

我们将实现 k-means 聚类、构建一个经过 EM 算法训练的高斯混合模型。

## 拉取仓库

```
git clone https://github.com/BeiYazi0/Unsupervised-learning
```

## 设置

创建并激活环境

```
conda create -n ai_env python=3.7
conda activate ai_env
```

安装依赖

```
cd Unsupervised-learning
pip install -r requirements.txt
```

## Jupyter

在`notebook.ipynb`中提供了进一步的说明。运行:

```
jupyter notebook
```
